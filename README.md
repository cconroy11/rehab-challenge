# vue-app

## Project setup
```
npm install
```

## Create .env in directory
```
touch .env
```

## Fill .env with neccesary values
```
VUE_APP_API = 'http://localhost:8080/api/'
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
